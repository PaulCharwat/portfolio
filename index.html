Technische Vorraussetzungen

Zu beginn der Vorlesung wurden uns hilfreiche Anwendungen und Tool gezeigt, die das Programieren und die Entwicklung von Webseiten erleichtern. Dazu gehören:
  • Virtual Studio Code, eine IDE (Integrated Development Environment) von Microsoft, die korrekten Syntax farblich hervorhebt und bei der Behebung von Fehlern hilft.
  • Notepad++, ein Textbearbeitungsprogramm für Windows, dass auch einen Syntax Highlighter besitzt.
  • BBedit, ein vergleichbares Programm für MacOS.
  • Außerdem ist die Web-Konsole des Browsers äußerst nützlich für die analyse und Fehlersuche des geschriebenen codes.
